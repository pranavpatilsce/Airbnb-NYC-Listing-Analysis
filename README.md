# Airbnb-NYC-Listing-Analysis
Using listing data from http://insideairbnb.com/get-the-data.html

> cd frontend
> - npm start
> - Press Ctrl+C to exit the localhost app

> cd service
> - source bin/activate
> - FLASK_APP=app.py flask run
> - Press Ctrl+C to exit the localhost app
> - deactivate (to deactivate the virtualenv)
